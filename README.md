Implementações dos Testes de Hipóteses vistos na cadeira de Inferência Estatística. Os seguintes testes foram implementados:
- Teste de Kolmogorov-Smirnov
- Teste de Shapiro-Wilk
- Teste X² de Bartlett
- Teste Z para média populacional (σ conhecido)
- Teste t de Student para média populacional (σ desconhecido)
